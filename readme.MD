## ColorSliderList  

### Android ui module to add a color slider like pointed block

### Usage

```
    //Creating the list
    ColorSliderList ColorSliderList = new ColorSliderList(this);
    //adding a colored view
    colorSliderList.addView(color);
    //showing the list
    colorSliderList.show();
```
